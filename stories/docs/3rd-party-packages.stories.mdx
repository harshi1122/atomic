# 3rd Party Packages

[Like RedwoodJS](https://redwoodjs.com/docs/introduction#technologies), Atomic makes choices for you in an effort to expedite building a stylish, modern user interface.
Many of these choices are particular npm packages. This document serves as an introduction to each of the major 3rd party packages Atomic provides your application.
It is **not** a form of documentation for these packages.
