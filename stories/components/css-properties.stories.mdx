# Custom CSS Properties

Define one or many [custom CSS properties](https://developer.mozilla.org/en-US/docs/Web/CSS/--*) from the comfort of your React application.

> `selector` can be any valid [CSS selector](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Selectors) â€” letting you scope property values to a very specific, and potentially dynamic, set of elements.

## CSSProperty

### Example

```tsx
import { CSSProperty } from '@locktech/atomic'

export const App = () => (
  <CSSProperty
    selector=":root"
    property="my-border"
    value="1px dashed #ff0000"
  />
)
```

### Parameters

| Property     	| Required     	| Default     	| Description                                                                  	|
|--------------	|--------------	|--------------	|------------------------------------------------------------------------------	|
| `selector`   	| `false`     	| `:root`      	| The CSS selector under which this property will be added. 	                  |
| `property`   	| `true`      	| `undefined`  	| The 'key' of the CSS property - which will be used to access it from styles. 	|
| `value`   	  | `true`      	| `undefined`  	| The property's value.                                                       	|

## CSSProperties

### Example

```tsx
import { CSSProperties } from '@locktech/atomic'

export const App = () => (
  <CSSProperties
    selector=":root"
    name="foo"
    properties={{ key: 'value' }}
  />
)
```

### Parameters

| Property     	| Required     	| Default     	| Description                                                                  	|
|--------------	|--------------	|--------------	|------------------------------------------------------------------------------	|
| `selector`   	| `false`     	| `:root`      	| The CSS selector under which this property will be added. 	                  |
| `name`    	  | `false`      	| `undefined`  	| An optional string which will be prefixed to each property.                  	|
| `properties` 	| `true`     	  | `undefined`  	| A `Record` of key-value pairs to be added as properties.                    	|
