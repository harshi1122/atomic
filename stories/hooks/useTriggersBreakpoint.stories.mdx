# useTriggersBreakpoint

A hook which returns a boolean when the corresponding [width media feature](https://developer.mozilla.org/en-US/docs/Web/CSS/@media/width) would be applied.

* `max` — Return `false` when the device's width is **greater than** the breakpoint's.
* `min` — Return `true` when the device's width is **greater than** the breakpoint's.

### Example

```tsx
import { useTriggersBreakpoint } from '@locktech/atomic'

export const Component = () => {
  const isAtLeastSm = useTriggersBreakpoint('sm') // returns `true` when the device's width is at least the breakpoint "sm"
  const isWiderThan400px = useTriggersBreakpoint('400px', 'max') // returns `false` when the device's width is less than 400px
  return ...
}
```

### Parameters

| Property     	| Required     	| Default     	| Description                                                                                          	 |
|--------------	|--------------	|--------------	|------------------------------------------------------------------------------------------------------	 |
| `bp`        	| `true`      	| `undefined`  	| A configured breakpoint, or an arbitrary width.                                                        |
| `variant`     | `false`      	| `'min'`     	| Which [feature variant](https://developer.mozilla.org/en-US/docs/Web/CSS/@media/width#syntax) is used to determine if the given breakpoint is triggered. |
